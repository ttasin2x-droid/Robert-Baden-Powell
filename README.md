# ⚜️ Robert Baden-Powell - Father of Scouting

A digital tribute dedicated to Lord Robert Baden-Powell (B.P.), exploring his complete biography, the rich history of the Scout movement, and his historic milestones. This is a fully responsive, interactive, and visually engaging web project built to showcase the true "Scouting Spirit."

🚀 **Live Website Link:** [Visit the Website Here](https://ttasin2x-droid.github.io/Robert-Baden-Powell/)

---

## ✨ Features

- **Modern Glassmorphism Design:** A premium UI featuring a seamless Day ☀️ and Night 🌙 mode toggle.
- **Dynamic Animations:** Includes a custom golden cursor, dynamic typing text, fireflies, and an interactive particle background canvas.
- **Interactive Timeline:** A visually appealing journey through B.P.'s life, covering major events like the Siege of Mafeking and the Brownsea Island Camp.
- **3D Book Section:** Interactive 3D tilt-hover effects on B.P.'s famous books with direct links to read or download PDFs.
- **Campfire Mode 🔥:** An immersive mode that dims the lights, adds a warm glow, and plays realistic campfire crackling sounds.
- **Scout Anthem 🎶:** A built-in audio player to play the official Scout music in the background.
- **Media Archive:** A Lightbox-powered historic image gallery and an embedded vintage video archive.
- **Scout Law & Promise:** An interactive accordion menu detailing the core principles of Scouting.

---

## 🛠️ Technologies Used

- **HTML5:** Semantic and clean markup.
- **Tailwind CSS:** Modern utility-first styling, Flexbox, Grid, and responsive design.
- **Vanilla JavaScript:** DOM manipulation, audio control, custom cursors, and interactive logic.
- **GSAP & ScrollTrigger:** For smooth, professional scroll-reveal animations.
- **Custom Local Fonts:** Hosted locally to ensure zero layout shifting and faster load times.

## 📂 Folder Structure

To run this project properly with all assets, ensure the following directory structure is maintained:

```text
📁 Robert-Baden-Powell
 ├── 📄 index.html        # Main HTML file
 ├── 📄 style.css         # CSS file for custom styling & animations
 ├── 📄 script.js         # JavaScript file for interactions
 ├── 📁 fonts/            # Custom fonts (bornomala.ttf, hind-regular.ttf, hind-bold.ttf)
 ├── 📁 photos/           # Historic images & book covers (hero.jpg, book1.jpg, etc.)
 ├── 📁 pdfs/             # Downloadable books (book1.pdf, book2.pdf, etc.)
 ├── 📁 music/            # Audio files (scoutmp.mp3, campfire.mp3)
 └── 📁 videos/           # Historic video clips (bpvid1.mp4, etc.)
```

---

## 🚀 How to Run Locally

1. Clone this repository or download it as a ZIP file.
2. Ensure all asset folders (`fonts`, `photos`, `pdfs`, `music`, `videos`) contain the correct files.
3. Open the `index.html` file in any modern web browser (Chrome, Edge, Safari, etc.).

*(Note: For the best experience and to avoid local CORS policy restrictions, it is recommended to run the project using a local server like the "Live Server" extension in VS Code).*
